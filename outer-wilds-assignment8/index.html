About the Game

A Space Mystery

Outer Wilds is an exploration game about a small solar system caught in a time loop. The player travels between planets, reads old messages, and slowly figures out what happened. The game is mostly about curiosity instead of fighting enemies or collecting upgrades.

Why I Chose It

I chose Outer Wilds because it makes exploring feel important. There is no single correct order for visiting the planets. Finding one clue usually gives the player an idea about where to look next, which makes the game feel like a mystery.
